En résumé, je n'ai peur de rien !

Mes caractéristiques

Développement mobile

Je maîtrise le développement hybride IOS/Android via Phonegap/Cordova, jQuery et BackboneJS. Je crée des serveurs de types NodeJS, contenant une partie sockets (socket.io) et modèles (BookshelfJS). Je gère aussi la plupart des fonctionnalités d'un téléphone.

Développement de sites web

Je suis capable de créer des sites web vitrines de style one-page. J'utilise différentes librairies pour réaliser des animations (parralaxes, animations sur-mesures, travail d'images).

Développement d'applications

J'ai réalisé des back-offices d'applications. J'utilise le framework PHP CakePHP ainsi que Bootstrap. J'ai également réalisé plusieurs outils comme un runner de serveurs NodeJS hébergé sous NPM ou encore des scripts de migrations de base de données différentes (SQL Server vers MySQL)

Responsive design

Toutes mes applications et sites réalisés ont été adaptés sur tout types de terminaux.

Qualité et organisation du code

J'ai pour habitude de développer de manière propre afin de rendre le code réutilisable. De plus, j'utilise essentiellement GIT afin de veiller à l'organisation de mon travail, de mettre en production et de gérer la maintenance.
